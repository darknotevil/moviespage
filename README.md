Коллекция плагинов для лампы

Можно подключить в лампу по ссылке на github pages.
https://darknotevil.github.io/lampa-plugins-collection/collections.js


- collections.js - изменяет раздел коллекции на прокручивающийся список избранное + пользовательские коллекции. Убирает чужие коллекции. Добавляет управление коллекцией в карточку фильма
- lme-slim.js - убирает лишние кнопки и эмодзи на карточке фильма, позволяет изменять порядок 
- unblock.js - снимает ограничения категорий (не снимает dmca)
- kinopoisk.js - сырой плагин. Должен открывать приложение кинопоиска с этим фильмом. Долгое нажатие - открывает сайт. Не работает с приложением кинопоиск тв на андроид.
